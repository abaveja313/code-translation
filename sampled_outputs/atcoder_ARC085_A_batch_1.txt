Explanation:

* `import java.util.Scanner;` imports the `Scanner` class from the `java.util` package, which is used to read input from the user.
* `public class Solution {` declares a public class called `Solution`.
* `public static void main(String[] args) {` declares a public static `main` method, which is the entry point for the program.
* `Scanner sc = new Scanner(System.in);` creates a new `Scanner` object called `sc` that reads input from the `System.in` stream.
* `int N = sc.nextInt();` reads the first integer from the input and assigns it to the variable `N`.
* `int M = sc.nextInt();` reads the second integer from the input and assigns it to the variable `M`.
* `int one = 100 * (N - M) + 1900 * M;` calculates the value of `one` using the formula from the Python code.
* `int count = 1 << M;` calculates the value of `count` using the bitwise left shift operator.
* `System.out.println(one * count);` prints the result of the multiplication.

Note: The `<<` operator is the bitwise left shift operator, which shifts the bits of the first operand to the left by the number of bits specified by the second operand. In this case, it is used to calculate the value of `count` by shifting the binary representation of `1` to the left by `M` bits.
