import java.util.Scanner;

class Solution {
  public static void main(String[] args) {
    Scanner scanner = new Scanner(System.in);
    int n = scanner.nextInt();
    int ans = n / 5;
    if (n % 5 == 0) {
      System.out.println(ans);
    } else {
      System.out.println(ans + 1);
    }
  }
}

Explanation:

1. The `import` statement is used to import the `Scanner` class from the `java.util` package.
2. The `Scanner` class is used to read input from the user. In this case, we create a new `Scanner` object called `scanner` and use it to read an integer input from the user.
3. The `n` variable is declared and initialized to the integer input read from the user.
4. The `ans` variable is declared and initialized to the result of dividing `n` by 5.
5. We use the `if` statement to check if `n` is divisible by 5. If it is, we print `ans`. Otherwise, we print `ans + 1`.
6. The `main` method is the entry point of the program, and it takes an array of strings as an argument. In this case, we don't need to use the `args` array, so we can omit it.
7. The `Solution` class is the driver class that contains the `main` method.

Note: In Java, the `int` type is used to represent integer values, and the `/` operator is used for integer division.
