    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        int k = scanner.nextInt();
        int x = scanner.nextInt();
        if (k * 500 >= x) {
            System.out.println("Yes");
        } else {
            System.out.println("No");
        }
    }
}

The Python code uses the `map()` function to convert the input strings to integers. In Java, we can use the `Scanner` class to read input from the user and convert it to integers. We then use the `if-else` statement to check if the condition `k * 500 >= x` is true, and print "Yes" or "No" accordingly.

        int x = sc.nextInt();
        if (k * 500 >= x) {
            System.out.println("Yes");
        } else {
            System.out.println("No");
        }
    }
}
